About CitizenAI

CitizenAI is an AI-powered chatbot designed to help citizens get quick responses to government-related queries. The platform also analyzes sentiment from user feedback to better understand public opinion.

Built with Flask, MongoDB, Hugging Face LLMs, and Tailwind CSS, CitizenAI enables transparent communication between governments and citizens using cutting-edge natural language technology.

Key Features

  • AI-Powered Responses: Get instant answers using IBM Granite AI models
  • Sentiment Analysis: Automatic feedback analysis to improve services
  • Dashboard Analytics: Visual insights into citizen feedback trends
  • Secure Authentication: Safe and secure user account management
  • Responsive Design: Works seamlessly across all devices

Technology Stack

  • Backend: Flask (Python web framework)
  • Database: SQLite with SQLAlchemy ORM
  • AI Model: IBM Granite 3.3-8b-instruct via Hugging Face
  • Sentiment Analysis: TextBlob for natural language processing
  • Frontend: HTML5, CSS3, and vanilla JavaScript
  • Charts: Chart.js for data visualization

Our Mission

We believe in making government services more accessible and responsive to citizen needs. By leveraging AI technology, we aim to bridge the communication gap between citizens and government institutions, making public services more efficient and user-friendly.
